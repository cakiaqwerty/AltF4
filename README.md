Dying in Freelancer mode cost you 50% of your Merces and any gear you brought with you.<br/>
Dying again in alerted territory will fail the campaign and lose all your freelancer tools.

The game treats Alt+F4 as a normal quit, meaning it will save the game prior closing.<br/>
I don't wanna lose my beloved Sieker, This script make designated key to force quit Hitman Freelancer (provided you are fast enough before you fully died) to prevent losing items.

Most players use SuperF4 https://github.com/stefansundin/superf4<br/>
